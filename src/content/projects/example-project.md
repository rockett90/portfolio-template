## Overview

This is an example project. Delete this file and drop your own Markdown files
into `src/content/projects/` — each one automatically becomes a card on the
home page and a full page at `/projects/<filename>`.

## Anything Markdown works

Use headings, **bold**, *italic*, [links](https://example.com), lists, tables,
and code blocks:

```js
console.log("hello from a code block");
```

> Block quotes render as a styled callout.
